# 📦 Sistema de Distribución - Proyecto SENA 2025

Sistema de gestión de distribución con módulos de productos, clientes, ventas, facturación y envíos.

## 🚀 Instalación Rápida

### Requisitos Previos
- **Node.js** v16 o superior ([Descargar](https://nodejs.org/))

### Pasos para ejecutar en otro equipo:

```bash
# 1. Abrir terminal en la carpeta del proyecto
cd ruta/a/la/carpeta/Proyecto1

# 2. Instalar dependencias
npm install

# 3. Iniciar el servidor
npm start
```

### 4. Abrir en el navegador:
```
http://localhost:4000
```

## 📁 Estructura del Proyecto

```
Proyecto1/
├── public/              # Archivos estáticos (HTML, CSS, JS)
│   ├── css/            # Estilos
│   ├── js/             # Scripts del cliente
│   └── *.html          # Páginas
├── routes/             # APIs del servidor
├── data/               # Datos XML
├── utils/              # Utilidades (cargador XML)
├── server.js           # Servidor Express
└── package.json        # Dependencias
```

## ⚙️ Funcionalidades Implementadas

### Módulo de Productos ✅
- ✅ Listado con búsqueda y filtros por categoría

- Distribución

## 🛠️ Tecnologías

- **Backend**: Node.js + Express
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Datos**: XML + Almacenamiento en memoria
- **Estilos**: CSS moderno con variables y responsive design
